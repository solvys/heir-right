# Sprint Brief: S6-T5 -- Paid/Manual Source Governance

Owner: TP
Parent: S6-ORCH
Dependencies: S6-T1, S6-T2, S6-T3, S6-T4

## Goal

Keep paid and manual research sources governed until the client approves access, storage, and automation.

## Included Scope

- Source access class taxonomy.
- IDI/Intelius/Ancestry/ForeWarn/VitalChek/PI as approval-gated sources.
- Door knock, neighbor research, code enforcement, and document request as manual tasks.
- Storage approval flags.
- Audit notes for manual observations.

## Excluded Scope

- Credential handling.
- Automated paid-source scraping.

## Acceptance

- No paid/manual source can be treated as automated by default.
- Operator tasks can represent approved manual work.
