# Sprint Brief: S7-T3 -- Report Renderer

Owner: TP
Parent: S7-ORCH
Dependencies: S7-T1, S7-T2

## Goal

Generate an internal completed lead report from the dossier and workflow facts.

## Included Scope

- Markdown/HTML report renderer.
- Workflow section order.
- Source-link display.
- Missing-data display.
- Offer/profit table.
- Draft/review status banner.

## Excluded Scope

- Final PDF/Word fidelity unless conversion dependency is selected.

## Acceptance

- One sample dossier renders into a complete internal report.
- Report clearly distinguishes facts, hypotheses, and missing items.
