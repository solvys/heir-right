# Sprint Brief: S8-T3 -- Follow-Up Task Model

Owner: TP
Parent: S8-ORCH
Dependencies: S8-T1

## Goal

Model follow-up cadence as tasks without live automation.

## Included Scope

- Call each lead 3 times task pattern.
- Voicemail/text follow-up tasks.
- Morning and afternoon follow-up windows.
- Multi-day/week retry state.
- Joshua escalation state.
- Multi-contact opportunity support.

## Excluded Scope

- Auto-dialing.
- Auto-SMS or email send.

## Acceptance

- Operators can see next follow-up tasks.
- Escalation to Joshua is visible.
