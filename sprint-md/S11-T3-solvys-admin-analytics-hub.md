# Sprint Brief: S11-T3 -- Solvys Admin Analytics Hub

## Goal

Plan the internal Solvys admin dashboard for cross-project analytics, monitoring, and control.

## Build Scope

- Project registry.
- Client shell registry.
- Analytics event inventory.
- Error/monitoring event inventory.
- Per-project usage, run, blocker, and adoption metrics.
- Solvys-1 deeper control requirements.
- Future Fintheon/Teams integration hook notes.

## Exclusions

- Final OpenPanel vs PostHog decision before the analytics evaluation is complete.
- Client-facing generic analytics UI.

## Acceptance

- Analytics layer can support all white-labeled shells.
- Solvys-1-specific control needs are called out separately.
- Event contract is provider-agnostic until PostHog/OpenPanel is selected.
