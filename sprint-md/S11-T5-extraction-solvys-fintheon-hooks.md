# Sprint Brief: S11-T5 -- Extraction + Solvys-1/Fintheon Hooks

## Goal

Define the extraction path from HeirRight MVP into a reusable Solvys shell, with later Solvys-1 and Fintheon integration in mind.

## Build Scope

- Extraction readiness checklist.
- Shared package/module boundary.
- Solvys-1 admin control requirements.
- Fintheon future Teams integration hook notes.
- Migration plan for future client shells.

## Exclusions

- Premature extraction before HeirRight MVP proves the shell pattern.
- Deep Fintheon Teams implementation in this sprint.

## Acceptance

- Reusable shell extraction has clear gates.
- Solvys-1 and Fintheon needs are represented without widening HeirRight MVP.
- Future client projects can start from the shell without inheriting HeirRight-specific UX.
