# Sprint Brief: S8-T4 -- Outreach-Ready Criteria

Owner: TP
Parent: S8-ORCH
Dependencies: S7, S8-T2

## Goal

Define when a lead may move from research/reporting into manual outreach.

## Included Scope

- Source evidence completeness.
- Family tree review state.
- Completed lead report state.
- Offer math review state.
- Script review state.
- Contact data review state.

## Excluded Scope

- Compliance decision itself.
- Live outreach automation.

## Acceptance

- Outreach-ready state is computed from review gates.
- Missing prerequisites produce visible blockers.
