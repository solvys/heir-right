# HeirRight Site V2 Prototype

This folder is the S10 website redesign scaffold.

- `index.html` presents three reviewable homepage directions.
- `index.html` also includes the S10-T3 copy/layout draft choices for review.
- `styles.css` carries the shared prototype system.
- `../../docs/HEIRRIGHT_SITE_COPY_LAYOUT_DRAFTS.md` is the plain-language draft packet.

Open `index.html` directly in a browser for review or serve the folder with any static server.
